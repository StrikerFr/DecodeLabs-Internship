Welcome to my portfolio repository for the **DecodeLabs Artificial Intelligence Internship 2026**. This repository serves as a centralized index showcasing the projects developed throughout the internship program, each exploring different facets of modern web engineering, artificial intelligence, and user experience design.

## 📋 About the Internship

The DecodeLabs AI Internship is a rigorous program focused on:
- **Hands-on AI Development**: Building end-to-end applications powered by state-of-the-art machine learning models.
- **LLM Integrations**: Leveraging models from OpenAI, Anthropic, and Groq (Llama) for natural language reasoning.
- **Modern Web Technologies**: Designing fluid, responsive frontends using React, TypeScript, and TailwindCSS.
- **Real-World Applications**: Translating complex models into practical, user-centric web tools.

## 🚀 Projects Showcase

### 🤖 DecodeBot
An AI-powered conversational assistant featuring multiple interaction modes, voice capabilities, and immersive user experiences. It is designed to demonstrate advanced chat completions, response streaming, and interactive prompt guidance.

### 📊 ClassifyAI
An intelligent classification platform capable of analyzing datasets, comparing machine learning models, and generating explainable insights. It provides automated insights, visual data distribution breakdowns, and model performance metrics.

*   **Branch:** [`Task-2-AryanGarg`](https://github.com/StrikerFr/DecodeLabs-Internship/tree/Task-2-AryanGarg)

### 🎬 MoviePlex
An AI-enhanced cinematic discovery platform designed to provide immersive movie exploration experiences. It uses a Three.js 3D spatial carousel, custom nebula shaders, and Groq Llama 3.3 server functions to match and explain movie recommendations based on user mood signals.

*   **Branch:** [`Task-3-AryanGarg`](https://github.com/StrikerFr/DecodeLabs-Internship/tree/Task-3-AryanGarg)
